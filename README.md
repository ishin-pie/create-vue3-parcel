# create-vue3-parcel
Initial vue + parcel + typescript

### Usage

   ```sh
   npm init vue3-parcel [project-name]
   ```

   or

   ```sh
   npm i create-vue3-parcel -g
   ```

   ```sh
   vue-parcel [project-name]
   ```
